# 🎓 Pratica+ | Apoio Escolar Personalizado

O **Pratica+** é uma plataforma web informativa e de agendamento desenvolvida para um projeto de explicações focado em alunos do 5º ao 12º ano no Porto. O site foi construído com foco em performance, design responsivo e facilidade de conversão para o utilizador final.

📍 **Link do Projeto:** [https://esteves1903.github.io/Pratica-site/](https://esteves1903.github.io/Pratica-site/)

---

## 🚀 Sobre o Projeto

O objetivo deste site é servir como a montra digital para o projeto Pratica+, fundado por estudantes do **ISEP** e da **FEP**. O site apresenta a missão da equipa, as disciplinas lecionadas (Matemática, Física, Economia, MACS e Inglês), tabelas de preços transparentes e uma integração direta para agendamento de sessões.

### Funcionalidades Principais:
- **Design Responsivo:** Otimizado para telemóveis, tablets e computadores.
- **Sistema de Agendamento:** Integração com a API do **Calendly** para marcação de sessões experimentais.
- **Call-to-Action (CTA):** Botões estratégicos para contacto direto via WhatsApp.
- **FAQ Interativo:** Secção de perguntas frequentes com lógica em JavaScript para melhor experiência de utilizador.
- **Interface Clean:** Design moderno focado na legibilidade e confiança académica.

---

## 🛠️ Tecnologias Utilizadas

Este é um projeto 100% **Front-end**, focado na rapidez de entrega e segurança dos dados do utilizador:

- **HTML5:** Estrutura semântica do site.
- **CSS3:** Estilização personalizada utilizando **Flexbox** e **CSS Grid**.
- **JavaScript (Vanilla):** Lógica de interatividade (Scroll suave, FAQ, integração Calendly).
- **Google Fonts:** Tipografia "Inter" para um aspeto moderno e legível.
- **GitHub Pages:** Alojamento e Deployment contínuo.
